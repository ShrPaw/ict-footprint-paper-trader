## How It Works (Architecture)
- Data feed: Hyperliquid public API (api.hyperliquid.xyz) — free, no auth needed
- Paper bots (bot:eth etc): PaperEngine — simulated orders locally
- Live bots (live:eth etc): HyperliquidEngine — real orders on testnet via ccxt
- Both use the same HyperliquidFeed for live candle data
- Strategy: DaytradeMode (1H ICT + footprint confluence)
- Polling interval: 3 seconds for live data

## To Run Live on Hyperliquid Testnet
1. Go to https://app.hyperliquid-testnet.xyz/
2. Create/connect wallet (MetaMask works)
3. Get testnet USDC from faucet
4. Export private key
5. Add to .env: HYPERLIQUID_PRIVATE_KEY=0x...
6. Run: npm run live:eth (or live:all)
