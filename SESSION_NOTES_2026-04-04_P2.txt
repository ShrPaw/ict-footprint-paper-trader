# Session Notes — 2026-04-04 Quant Research

## Session Summary

Ran institutional-grade Quant Research System (ROS) to determine if a statistically valid edge exists in the ict-footprint-paper-trader system.

## What Was Done

### Phase 1: Data Validation
- Acquired 691,200 candle-hours from Binance public API
- BTC/ETH/SOL/XRP × 1m/5m/15m/1h timeframes
- Coverage: Oct 2024 → Apr 2026 (18 months)
- Multiple market regimes covered: bull (Q4 2024), bear (Q1-Q2 2025), sideways (Q3-Q4 2025)

### Phase 2: Event-Level Edge Discovery
- Tested 9 high-recall event types across all 4 assets independently
- 288 total statistical tests (9 events × 4 horizons × 4 assets × 2 timeframes)
- Events: session opens, displacement candles, VWAP deviation, volatility expansion, liquidity sweeps, consecutive candles
- Result: VWAP deviation (>2%) emerged as the only cross-asset candidate
- 8 edge candidates found, validated against random baselines (5,000 Monte Carlo sims)

### Phase 3: Entry Validation
- All VWAP deviation tests survived Bonferroni correction
- 5 of 11 passed "not tail-driven" filter
- Below-VWAP deviation showed strongest signal (BTC +1.57%, SOL +0.87%, XRP +0.85% over 240 bars)

### Phase 4-5: Time Modeling + Loss Forensics
- Optimal horizon: 18-24h (edge increases with holding time)
- Below-VWAP direction consistently outperforms above-VWAP
- BUT: MAE hits before MFE in >50% of cases
- Catastrophic outcomes: worst cases -14% to -18%

### Phase 2.5: Stress Testing — EDGE REJECTED
Six adversarial tests applied:

| Test | BTC | SOL | XRP |
|------|-----|-----|-----|
| Regime Segmentation | FAIL | FAIL | PASS |
| Event Clustering | PASS | PASS | PASS |
| Entry Delay Sensitivity | FAIL | FAIL | FAIL |

**Verdict: CASE B — FRAGILE/FALSE EDGE**

## Key Learnings

1. **t-stats can lie**: Original t-stats of 3.6-6.1 were artifacts of cluster inflation and regime pooling
2. **Cluster dependence is deadly**: 72-76% of events were in clusters (mean size 4.6-5.4), inflating effective sample by ~2.3x
3. **Random baseline is necessary but not sufficient**: Passing random baseline doesn't mean the edge is real — it just means it's not pure noise
4. **Directional adversarial testing is critical**: For SOL, above-VWAP actually outperformed below-VWAP — the detected edge was the WRONG direction
5. **Mean reversion edges in crypto are fragile**: They tend to be regime-dependent and timing-sensitive
